Report a Bug

Have you found a bug or mistake on Tobira Study Resources or just want to submit general feedback? If so, you can let us know by opening a new issue! Please read the information provided below to learn how to submit your report or feedback.

Note: We utilize GitHub's "Issues" tracker to collect user feedback. If you're not familiar with this system you can read about it here. A GitHub account is required to submit new issues. If you don't have one you can register an account here (optional, see contact methods for alternatives).

Reporting a Technical Bug

Is Tobira Study Resources not working properly on your system? If so, please select your preferred contact method and provide the following information.

  • Description of the bug
  • Steps to replicate the bug (if possible)
  • Browser(s) the bug occurred in
  • Operating System
  • Were you using the Online or Offline version?

Note: If you are able to fix the bug you may submit the fix as a pull request instead.

We want to make sure that TSR functions properly for everyone, so we greatly appreciate your report and will try to resolve your issue as soon as humanly possible!

Contact Methods

  • Open a new issue on GitHub
  • PM me on Reddit
  • DM me on Twitter

Reporting a Mistake

Have you noticed a mistake on Tobira Study Resources, such as a spelling error or erroneous answer? If so, please select your preferred contact method and provide the following information.

  • Link to the page that contains the mistake(s)
  • Erroneous content
  • Correction of the content

Note: If you like, you can fix the mistake yourself and submit the correction as a pull request instead.

We want to make sure that students receive accurate information, so we greatly appreciate your help in ensuring that!

Contact Methods

  • Open a new issue on GitHub
  • PM me on Reddit
  • DM me on Twitter

General Feedback

Does your query not meet the conditions listed above? No problem, we'd still love to hear from you! If you have a suggestion, complaint, or comment, feel free to open a new Issue on GitHub or contact me on Reddit/Twitter.

  • Open a new issue on GitHub (requires GitHub account)
  • PM me on Reddit
  • DM me on Twitter

Any and all feedback is appreciated; we look forward to hearing from you!
